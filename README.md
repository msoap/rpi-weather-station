# rpi-weather-station

RPI a simple weather station

## Hardware

- Raspberry Pi 1
- BME280 sensor
- SH1106 OLED display, 128x64 screen, 4 wire SPI

## Build for Raspberry Pi 1*

```sh
make build-rpi
```
